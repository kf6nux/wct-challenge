Thank you for trying this coding challenge!

To begin, let's set up your dependencies:
bower install
npm install -g web-component-tester

You will be writing tests for:
/myelement.html

You can write tests in:
/test/basic-test.html

You can run your tests with:
wct

Don't take a lot of time on this, and feel free to share your thoughts on the exercise with your submission.

Have fun!
